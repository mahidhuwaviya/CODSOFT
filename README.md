CODSOFT Java Development Internship
This repository contains the tasks completed during my 30-day internship at Codsoft. The projects range from console-based utilities to full-stack web applications and GUI systems.

🛠️ Tech Stack Used
Java (JDK 17 or higher)

Spring Boot (for Backend API)

JSP & Servlets (for Web Tasks)

Swing (for Desktop GUI)

Maven (Dependency Management)

Postman (API Testing)

🚀 How to Run the Projects
Task 1: [Guess The Number Game] (Web Application)
Description: A web-based project using JSP and CSS.

How to Run:

Open the project in an IDE (IntelliJ or Eclipse).

Configure a local server (e.g., Apache Tomcat 9/10).

Run the project on the server.

Access the UI at http://localhost:8080/[ProjectName].

Task 2: [Student Grade Calculator] (Console Application)
Description: A logic-based Java program running entirely in the terminal.

How to Run:

Navigate to the Task2 folder.

Run the StudentGradeCalculator.java file.

Follow the prompts in the console/terminal.

Task 3: [ATM interface] (Web Application - Basic)
Description: A server-side project using JSP without external styling.

How to Run:

Deploy the project onto an Apache Tomcat server.

Run the application.

The application will open in your default browser.

Task 4: [CURRENCY CONVERTER] (Spring Boot API)
Description: A backend-focused project built with Spring Boot.

How to Run:

Ensure you have Maven installed.

Run the application as a Spring Boot App.

Testing: Since there is no frontend, use Postman to send requests (GET/POST) to http://localhost:8080/api/....

Task 5: [STUDENT MANAGEMENT SYSTEM] (Desktop GUI)
Description: A desktop-based application using Java Swing.

How to Run:

Open the Task5 source folder.

Locate the file: StudentSystemInterface.java.

Run this specific class to launch the graphical user interface (GUI).
